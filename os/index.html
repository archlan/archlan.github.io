ArchLan is an Arch-based distribution aiming for a minimal number of packages while still being fully customizable and powerful.

Download ArchLan

A new experience for you.

ArchLan aims to be minimalistic in the number of packages while still providing tips and content to customize your machine. The ArchLan ThemeSwitch enables you to easily change the apperance of the desktop, just like on MacOS and Windows.

Reliable.

ArchLan is a minimal and robust distribution using i3wm to make it able to run on most computers, even the oldest or least powerful ones.

Powerful.

Brought with OEM support for the AUR, ArchLan allows you to do whatever you want to do on your machine, using the power of Linux.

Private.

Your privacy is valued, that's why absolutely no personal data is collected. ArchLan is an Open Source project which means that anyone can verify the security and integrity of the source code.

A gigantic library of Software.

ArchLan is shipped with multiple software repositories allowing you to install a multitude of various applications. There is something for all tastes and uses.

Multi boot.

You can install ArchLan beside Windows or macOS to keep your files and apps safe and sound. You can choose which OS to boot each time your machine starts.

Personalization.

ArchLan is based on Arch Linux, which means you can customize your ArchLan from top to bottom to make it suit your needs and preferences. You can even customize the source code.

Works with all of your files.

The LibreOffice suite can be installed on ArchLan, which allows you to create and edit Microsoft Office files. You no longer have to worry about your presentation being corrupted or losing your text document report.

Common users questions.

Am I able to try out ArchLan before installing it on my machine?
Of course. You can boot the ArchLan USB drive and run the Live environment for as long as you wish to. In order to install, simply start the application "Welcome", and follow the on-screen instructions.

What is ArchLan cost?
ArchLan is currently free and will forever be free.

I'm afraid my computer won't work with ArchLan...
Your machine most certainly is able to have ArchLan installed on it. You can verify it meets the system requirements. Most machines can run ArchLan, as long as they're able to boot.

How can I know every apps I use will work?
In addition to all of the Arch Linux applications panel, ArchLan can execute .AppImage files and any file compatible with Linux. The official ArchLan software repository also provides some additional packages.

How long is the installation process?
Depending on your network connection speed and your computer hardware, the ArchLan installation process takes around 10 to 30 minutes.
